In order to keep things simple for status I am placing only
the source files within the folder/directory when possible.
This avoids filling the repository with compiler artifacts.

This means of course, the source files must be intoduced to 
the IDE (Visual Studio) as a new project, built & run in 
order for it to execute.
 
The build generates compiler files which are many and these 
files are not necessary to save in a repository.
